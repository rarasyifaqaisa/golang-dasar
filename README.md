# Golang Dasar Tutorial

This repository contains basic Go programming examples and exercises. Each folder represents different fundamental concepts in Go programming.

## Table of Contents

1. Hello World
   - Basic introduction to Go programming
   - Print statements and program structure

2. Variables
   - Variable declaration and initialization
   - Constants
   - Basic data types

3. Data Types
   - Numbers (int, float)
   - Strings
   - Boolean
   - Arrays and Slices

4. Operators
   - Arithmetic operators
   - Comparison operators
   - Logical operators

5. Control Flow
   - If/else statements
   - Switch statements
   - Loops (for, while)

## Prerequisites

- Go version 1.16 or higher
- Basic understanding of programming concepts

## How to Run

1. Make sure Go is installed on your system
2. Navigate to specific folder
3. Run the program using:
```
go run main.go
```
